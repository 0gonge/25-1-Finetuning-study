## ✏️ How to PR
1. `o11y-basic-study` 레포를 fork한다.
2. fork한 레포에서 `week1` 브랜치를 생성한다.
3. `./week1/suhyen` 폴더를 생성한 후 해당 주차에 공부한 내용을 커밋한다.
4. PR 템플릿에 따라 PR을 작성하며 Reviewers로 `@suhyenim`을 태그한다.

## ✏️ How to Study
1. 기본 학습 (주 1시간)
   - 공식 문서 학습
   - 코드 실행 및 결과 공유

2. 심화 학습 (주 3시간)
   - 개인 실험 및 분석
   - HuggingFace & Wandb 활용
   - 트러블 슈팅 기록

3. 대면 스터디
   - 학습 내용 공유 (30분-1시간)
   - 다중 GPU 실습 (30분-1시간)
   - 팀 식사 필수
