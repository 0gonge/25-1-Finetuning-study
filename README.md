## ✏️ How to Study
1. 기본 학습 (주 1시간)
   - 공식 문서 학습
   - 코드 실행 및 결과 공유

2. 심화 학습 (주 3시간)
   - 개인 실험 및 분석
   - HuggingFace & Wandb 활용
   - 트러블 슈팅 기록

3. 대면 스터디
   - 학습 내용 공유 (30분-1시간)
   - 다중 GPU 실습 (30분-1시간)
   - 팀 식사 필수


## 📋 Prerequisites

- 높은 참여도와 열정
